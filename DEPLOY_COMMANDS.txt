# EASY DEPLOYMENT COMMANDS

## Step 1: Create your GitHub repository
Go to: https://github.com/new
- Repository name: llmtesting
- Public
- UNCHECK all boxes (no README, no .gitignore, no license)
- Click "Create repository"

## Step 2: Run this single command to deploy:
git push -u origin HEAD

If that doesn't work, run these commands one by one:
git remote add origin https://github.com/Mohansaina/llmtesting.git
git push -u origin HEAD

## Your website will be available at:
https://Mohansaina.github.io/llmtesting

## To enable GitHub Pages:
1. Go to your repository Settings
2. Click "Pages" in the left sidebar
3. Under "Source", select "GitHub Actions"
4. Click "Save"
